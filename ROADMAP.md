# PromptDNA Roadmap

This is the public roadmap for PromptDNA.  
Timelines may shift, but direction stays consistent.

## Now (Live)
- Image-to-prompt extraction
- Prompt remixing and reuse
- Credit-based rendering system
- Creator-first workflows

## Next
- Video frame prompt extraction
- Batch prompt analysis
- Prompt versioning
- Exportable prompt formats

## Later
- Team workspaces
- Prompt provenance tracking
- API access
- Enterprise plans
